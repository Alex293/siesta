Options which control the behavior of a Resource.

    Time before valid data is considered stale by Resource.loadIfNeeded().

  • responseTransformers

    A sequence of parsers to be applied to responses.

    You can add custom parsing using:

    responseTransformers.add(MyCustomTransformer())
responseTransformers.add(MyCustomTransformer(), contentTypes: ["foo/bar"])

    By default, the transformer sequence includes JSON and plain text parsing. You can remove this default behavior by clearing the sequence:

    responseTransformers.clear()

  • Builder

    Holds a mutable configuration while closures passed to Service.configure(...) modify it.

    The reason that method doesn’t just accept a closure with an inout param is that doing so requires a messy flavor of closure declaration that makes the API much harder to use:

    configure("/things/​*") { (inout config: Configuration) in config.retryTime = 1 }

    This wrapper class allows usage to instead look like:

    configure("/things/​*") { $0.config.retryTime = 1 }
